# Black Myth: Wukong 🐵

> Black Myth: Wukong is a highly anticipated action-adventure game created by a Chinese game development team. Set against the backdrop of Journey to the West, it reimagines the classic story with visually stunning and immersive gameplay experiences.

## Game Edition Overview

### 1. Digital Standard Edition 💻
- Contains the base game

### 2. Digital Deluxe Edition 🎶
- Includes the base game
- Additional in-game content and a curated digital soundtrack
- Weapon: "Copper Cloud Staff" 🏹
- Costume set: "Hundred Performances Nuo Mask, Hundred Performances Coin-Lined Robe, Hundred Performances Hand Guards, Hundred Performances Hanging Leg Guards" 🛡️
- Trinket: "Wind Chime" 🎐

### 3. Physical Deluxe Edition 📦
- Contains the Digital Deluxe Edition activation code and a series of physical collectibles
- Collector's tin box, 1:1 golden headband replica, "Chaos Peach Orchard" scroll painting, Wind Chime necklace, Thunder Chisel ring, Golden Crow badge, stamps and postcards, and a warranty certificate

### 4. Physical Collector's Edition 🏆
- Contains the Digital Deluxe Edition activation code and exclusive physical collectibles
- Collector's tin box, 1:6 poseable protagonist action figure "Face Your Destiny," "Heart Sutra Received" scroll painting, Wind Chime necklace, Thunder Chisel ring, Golden Crow badge, stamps and postcards, and a warranty certificate

## Game Features ✨

- **Stunning Visuals**: Built on Unreal Engine, delivering exceptionally high graphic quality and detail.
- **Combat Experience**: A deeply layered action system with highly challenging combat design.
- **Cultural Representation**: Recreates the myths and legendary stories of traditional Chinese culture.
